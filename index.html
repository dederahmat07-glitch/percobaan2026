Deteksi Wajah Menggunakan MediaPipe Face Mesh

Aplikasi ini menggunakan teknologi MediaPipe Face Mesh untuk mendeteksi bentuk wajah secara real-time melalui kamera PC. Sistem membaca koordinat wajah dan menampilkan jaringan titik landmark yang menggambarkan struktur mata, hidung, bibir, dan kontur wajah.

Face Mesh merupakan teknologi Computer Vision yang banyak digunakan untuk aplikasi augmented reality, filter wajah, analisis gerakan wajah, serta berbagai aplikasi kecerdasan buatan berbasis kamera. Seluruh proses berjalan langsung pada browser pengguna.
