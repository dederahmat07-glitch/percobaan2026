Deteksi Wajah dengan MediaPipe Face Mesh

Aplikasi ini menggunakan teknologi Face Mesh dari MediaPipe untuk mendeteksi struktur wajah secara real-time melalui kamera PC. Sistem mampu membaca hingga 468 titik koordinat wajah yang menggambarkan bentuk mata, hidung, bibir, rahang, dan kontur wajah.

Teknologi Face Mesh banyak digunakan dalam pengembangan aplikasi Augmented Reality (AR), filter wajah digital, analisis ekspresi, serta berbagai aplikasi Computer Vision modern. Seluruh proses dilakukan langsung pada perangkat pengguna melalui browser.
